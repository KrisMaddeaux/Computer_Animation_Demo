Motion Capture Animation C++ Demo 

• Tech demo that showcases my strong understanding of animation programming by 
  implementing motion capture skeleton animations using forward kinematics loaded from a BVH file. 
• Developed using C++, OpenGL, and GLSL shaders 

Instructions: 
-To run demo open 'Motion_Capture_Animation_Demo.exe'

-Press WASD keys to move around scene

-Press and hold any mouse button while moving mouse to rotate camera
